

# 1. Crear proyecto con Vite + React

npm create vite@latest

Luego:

cd nombre-del-proyecto
npm install
npm run dev

Durante la creación, eliges:

Framework: React

Variant: JavaScript o TypeScript

---

# 2. Instalar React Router

Ya dentro del proyecto:

npm install react-router-dom

Resumen rápido

npm create vite@latest   # crear proyecto
cd mi-app
npm install              # instalar dependencias
npm run dev              # arrancar Vite
npm install react-router-dom  # instalar router
